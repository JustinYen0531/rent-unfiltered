# Ch. 10: Capital Markets and the Pricing of Risk 資本市場與風險定價
## 🎯 章節核心問題
> **如何科學地定義與衡量投資風險？為什麼僅僅承擔高風險並不一定能獲得高回報？金融市場定價只對哪一種風險進行補償？**
>
> **核心解答：**
> 投資的風險通常以歷史回報率的**標準差（波動度）**來衡量。風險分為兩類：**非系統性風險**（公司特有風險）與**系統性風險**（市場共同風險）。由於投資人可以透過無成本的**分散投資 (Diversification)** 來消除非系統性風險，因此市場定價不會對其進行任何補償。資產的風險溢酬完全取決於其無法被分散的**系統性風險**。
---
## 📌 一、 歷史報酬率與風險衡量 (Historical Returns and Risk)
### 1. 實現回報率 (Realized Return)
在時間 $t$ 到 $t+1$ 期間，持有某證券的總回報率（包含股利與資本利得）：
$`R_{t+1} = \frac{Div_{t+1} + P_{t+1} - P_t}{P_t} = \frac{Div_{t+1}}{P_t} + \frac{P_{t+1} - P_t}{P_t} = \text{Dividend Yield} + \text{Capital Gain Yield}`$
### 2. 平均實現回報率 (Average Return)
$`\bar{R} = \frac{1}{T} \sum_{t=1}^{T} R_t`$
### 3. 歷史風險衡量：變異數與標準差
標準差（波動度 Volatility）代表報酬率偏離平均值的程度，是衡量總風險的最常用統計指標。
- **歷史變異數 (Variance / $Var$)：**
	$`Var(R) = \sigma^2 = \frac{1}{T-1} \sum_{t=1}^{T} (R_t - \bar{R})^2`$
- **歷史標準差 (Standard Deviation / $\sigma$ / Volatility)：**
	$`Volatility = \sigma = \sqrt{Var(R)}`$
---
## 📌 二、 個別證券的預期回報與風險 (Individual Securities)
當我們面向未來時，我們會預測不同的經濟狀態（State of Economy）：
- **期望回報率 (Expected Return / $E[R]$)：**
	$`E[R] = \sum_{i=1}^{S} P_i \times R_i`$
	其中 $P_i$ 為狀態 $i$ 發生的機率，$R_i$ 為在狀態 $i$ 下的報酬率。
- **期望變異數 ($Var(R)$)：**
	$`Var(R) = E[(R - E[R])^2] = \sum_{i=1}^{S} P_i \times (R_i - E[R])^2`$
---
## 📌 三、 投資組合與風險分散 (Portfolios & Diversification)
### 1. 投資組合權重 (Portfolio Weights)
- 投資組合中各資產占總資金的比例：
	$`w_i = \frac{\text{Value of Asset } i}{\text{Total Portfolio Value}}`$
	其中所有權重相加必須等於 1：$`\sum w_i = 1`$。
### 2. 投資組合預期報酬率 (Expected Return of a Portfolio)
- 投資組合的期望回報是各個資產期望回報的加權平均值：
	$`E[R_p] = \sum w_i \times E[R_i]`$
### 3. 分散投資 (Diversification) 的本質
- **非系統性風險 (Unsystematic / Idiosyncratic / Firm-Specific Risk)：**
	- 僅影響單一公司或行業的風險（如新藥研發失敗、公司爆發財務醜聞）。
	- **特徵：** 透過將多種不同行業的股票打包成投資組合，這些個別事件的負面影響會被其他公司的正面事件所抵消。因此，**非系統性風險可以被消除**。
- **系統性風險 (Systematic / Market / Common Risk)：**
	- 影響整個經濟體的風險（如央行升息、全球通膨、戰爭、流行病）。
	- **特徵：** 無論你如何分散投資，所有公司都會受到波及。因此，**系統性風險無法被消除**。
---
## 📌 四、 風險定價與市場均衡 (Pricing Risk)
為什麼僅承擔系統性風險能獲得高回報，而承擔非系統性風險卻沒有補償？
- **無套利與分散化邏輯：**
	- 投資人不需要花費成本，只要購買大盤指數基金（如 S&P 500），就可以完全免除所有非系統性風險。
	- 如果市場對非系統性風險支付額外回報，這將是一個巨大的套利機會。
	- **金科玉律：**
		> **「市場不會為可以被無成本分散的風險（非系統性風險）支付任何風險補貼 (Risk Premium)。」**
- **結論：**
	- 資產的**總風險（標準差）**並不直接決定其期望回報率。
	- 決定資產回報率的是其所包含的**系統性風險的大小**。

## 📌 五、 經典例題與解題地圖 (Practice Questions)
### 例題 1：實現回報率計算
> **題目：** 你在年初以 50 元買入某股票，該公司在年中派發了 2.0 元的現金股利，年底時你以 53 元將股票賣出。請問你這一年的實現回報率是多少？
- **解答流程：**
	- 遞延收益率 = $2.0 / 50 = 4\%$
	- 資本利得收益率 = $(53 - 50) / 50 = 6\%$
	- 總回報率：
	  $`R = \frac{2.0 + (53 - 50)}{50} = \frac{5.0}{50} = 10\%`$
- **答案：** 實現回報率為 **10%**。
---
### 例題 2：歷史平均與標準差計算 (⚠️ 考試必考統計)
> **題目：** 某資產過去 4 年的歷史報酬率分別為 15%、-5%、10%、20%。請問其歷史平均報酬率與標準差（波動度）各是多少？
- **解答流程：**
	1. **平均報酬率：**
	   $`\bar{R} = \frac{15\% - 5\% + 10\% + 20\%}{4} = \frac{40\%}{4} = 10\%`$
	2. **計算偏差平方和：**
	   - 年 1：$(0.15 - 0.10)^2 = 0.0025$
	   - 年 2：$(-0.05 - 0.10)^2 = 0.0225$
	   - 年 3：$(0.10 - 0.10)^2 = 0$
	   - 年 4：$(0.20 - 0.10)^2 = 0.0100$
	   - 總和 = $0.0350$。
	3. **歷史變異數 (除以 T - 1)：**
	   $`Var = \frac{0.0350}{4 - 1} = \frac{0.0350}{3} \approx 0.011667`$
	4. **歷史標準差：**
	   $`\sigma = \sqrt{0.011667} \approx 10.80\%`$
- **答案：** 平均回報為 **10.0%**；標準差為 **10.80%**。
---
### 例題 3：期望值與期望標準差
> **題目：** 預測明年經濟狀況有三種：繁榮 (機率 30%，回報 25%)、正常 (機率 50%，回報 10%)、衰退 (機率 20%，回報 -15%)。計算該股票的預期回報與變異數。
- **解答流程：**
	1. **期望回報率 $E[R]$：**
	   $`E[R] = 0.30 \times 25\% + 0.50 \times 10\% + 0.20 \times (-15\%) = 7.5\% + 5.0\% - 3.0\% = 9.5\%`$
	2. **期望變異數 $Var(R)$：**
	   $`Var(R) = 0.30 \times (0.25 - 0.095)^2 + 0.50 \times (0.10 - 0.095)^2 + 0.20 \times (-0.15 - 0.095)^2`$
	   $`Var(R) = 0.30 \times (0.155)^2 + 0.50 \times (0.005)^2 + 0.20 \times (-0.245)^2`$
	   $`Var(R) = 0.30 \times 0.024025 + 0.50 \times 0.000025 + 0.20 \times 0.060025 = 0.0072075 + 0.0000125 + 0.0120050 = 0.019225`$
	3. **標準差：**
	   $`\sigma = \sqrt{0.019225} \approx 13.87\%`$
- **答案：** 預期回報率為 **9.5%**；標準差為 **13.87%**。
---
### 例題 4：投資組合回報計算
> **題目：** 你投資了 3 萬元在股票 X (預期回報 12%)，另投資 7 萬元在股票 Y (預期回報 8%)。請問該投資組合的預期回報率是多少？
- **解答流程：**
	1. 總資產 = $30,000 + 70,000 = 100,000$ 元。
	2. 權重：$w_X = 0.30, w_Y = 0.70$。
	3. 套用公式：
	   $`E[R_p] = 0.30 \times 12\% + 0.70 \times 8\% = 3.6\% + 5.6\% = 9.2\%`$
- **答案：** 投資組合預期回報率為 **9.2%**。

## 📌 六、 程式化財務思維：JavaScript 風險統計計算器
我們可以使用程式碼來自動讀取一組歷史報酬率數據，並進行平均數、變異數與波動度的統計計算：
```javascript
/**
 * 計算歷史數據統計指標
 * @param {Array<number>} returns - 歷史報酬率數組 (如 [0.15, -0.05, 0.10, 0.20])
 */
function calculateHistoricalStatistics(returns) {
  const t = returns.length;
  if (t < 2) throw new Error("數據至少需要兩期以上");
  
  // 1. 計算平均回報
  const average = returns.reduce((sum, val) => sum + val, 0) / t;
  
  // 2. 計算變異數 (T - 1 樣本無偏估計)
  const sumSquaredDiff = returns.reduce((sum, val) => {
    return sum + Math.pow(val - average, 2);
  }, 0);
  const variance = sumSquaredDiff / (t - 1);
  
  // 3. 計算標準差
  const volatility = Math.sqrt(variance);
  
  return {
    average: average,
    variance: variance,
    volatility: volatility
  };
}

// === 統計測試 ===
const historicalData = [0.15, -0.05, 0.10, 0.20];
const stats = calculateHistoricalStatistics(historicalData);

console.log(`平均回報率: ${(stats.average * 100).toFixed(2)}%`); // 10.00%
console.log(`歷史變異數: ${stats.variance.toFixed(6)}`);
console.log(`標準差(波動度): ${(stats.volatility * 100).toFixed(2)}%`); // 10.80%
```

### 2. 本章三大考試/常考觀念
1. **除以 T-1 還是 T：** 計算歷史樣本變異數與標準差時，分母永遠是 $T-1$（自由度調整）；而計算前瞻性概率分佈變異數時，直接乘上概率即可，不需要減 1。
2. **總風險不等於定價風險：** 許多學生誤以為波動度高的股票收益就一定高。事實上，若一檔股票波動度高但全是「非系統性風險」（如單一研發型生技股），它的定價回報率可能僅僅等於無風險利率，因為它的風險可以被輕易分散掉。
3. **分散投資的極限：** 當投資組合的股票數量增加時，組合的波動度會迅速下降。然而，它不會降到 0，而是會收斂到一個極限值，這個極限值就是市場整體的系統性風險水平。
