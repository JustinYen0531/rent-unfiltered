# Ch. 7: Investment Decision Rules 投資決策法則
## 🎯 章節核心問題
> **公司面臨多個投資專案時，如何挑選出能創造最高價值的專案？為什麼被廣泛使用的 IRR 指標有時會給出完全錯誤的決策建議？在資金有限的情況下，該如何取捨？**
>
> **核心解答：**
> **NPV 法則**是決策的最高指導原則（NPV > 0 即接受）。雖然**內部報酬率 (IRR)** 非常直觀，但當專案規模不同、現金流方向改變、或專案互斥時，IRR 會產生嚴重的決策偏差。當公司面臨資本約束（資金有限）時，應使用**獲利能力指數 (PI)** 法則來最大化每元預算所創造的價值。
---
## 📌 一、 NPV 決策法則 (The NPV Rule)
- **核心法則：** 接受所有淨現值為正 ($NPV > 0$) 的獨立專案；拒絕所有淨現值為負 ($NPV < 0$) 的專案。
- **互斥專案 (Mutually Exclusive Projects)：** 如果多個專案只能選擇其一，應選擇 **NPV 最高者**，而非 IRR 最高者。
- **財務學共識：** NPV 是唯一在任何情況下都絕對正確的價值衡量工具。
---
## 📌 二、 內部報酬率法則與其三大致命缺陷 (IRR Rule & Its Pitfalls)
### 1. 內部報酬率 (Internal Rate of Return, IRR)
- **定義：** 使專案淨現值剛好等於零 ($NPV = 0$) 的折現率。
- **IRR 法則：** 若專案的 $IRR > \text{Cost of Capital (r)}$，則接受該專案。
### 2. IRR 的三大致命缺陷
儘管 IRR 在實務上極受經理人喜愛，但在以下情況下，IRR 法則會失效：

<tr>
<td>缺陷類型 (Pitfall)</td>
<td>發生情境 (Scenario)</td>
<td>為何 IRR 失效？ (Mechanism)</td>
</tr>
<tr>
<td>**1. 投資 vs. 融資問題 (Borrowing vs. Lending)**</td>
<td>現金流先流入、後流出（如借貸專案）。</td>
<td>此時 NPV 隨折現率上升而**上升**。IRR 法則會顛倒：我們應接受 $IRR < r$ 的融資，但傳統 IRR 卻會要求高 IRR，導致做出虧損決策。</td>
</tr>
<tr>
<td>**2. 多重 IRR 問題 (Multiple IRRs)**</td>
<td>專案的現金流正負號在存續期內**變更超過一次**（例如期末需要大筆拆除/環保清理費用）。</td>
<td>根據代數原理，現金流正負號變更幾次，就可能存在幾個不同的 IRR。當出現多個 IRR 時，無法與資本成本 $r$ 進行單一比較。</td>
</tr>
<tr>
<td>**3. 規模與時間衝突 (Scale & Timing Problems)**</td>
<td>互斥專案的投資規模相差巨大，或者現金流回流的時間截然不同。</td>
<td>IRR 是一個**百分比回報率**，忽略了專案的**絕對金額規模**。一個 100% 回報但投資 1 元的專案，絕對不如一個 10% 回報但投資 100 萬的專案。同理，IRR 也會偏好早期回收資金的專案，即使晚期高回報的專案 NPV 更高。</td>
</tr>
</table>
- **再投資利率假設差異：**
	- IRR 假設專案產生的中間現金流可以**以 IRR 的利率**進行再投資。這在實務上極不現實（高 IRR 專案很難找到同等高回報的再投資項目）。
	- NPV 則合理地假設所有中間現金流可以**以資金成本 $r$** 進行再投資。
---
## 📌 三、 回收期間法 (Payback Period Rule)
- **定義：** 計算需要多少時間，專案的累計現金流入才能回收初始投資成本。
- **法則：** 若回收時間小於公司設定的截止期限，則接受。
- **缺點：**
	1. 忽略了**貨幣的時間價值**（除非使用折現回收期間法）。
	2. **完全忽略了回收期之後的所有現金流**（例如一個 3 年收回成本但第 4 年能賺 10 億的專案，可能因為截止期限是 2 年而被拒絕）。
	3. 截止期限的設定是主觀且無理論支持的。
---
## 📌 四、 資金約束與獲利能力指數 (Capital Budgeting under Constraint)
當公司擁有許多正 NPV 專案，但總預算限制為 100 萬元（資本配額 Capital Rationing）時，我們無法單純選擇 NPV 最高的專案，而必須考慮資金的使用效率。
### 1. 獲利能力指數 (Profitability Index, PI)
- **定義：** 每投入 1 元初始資金所能創造的淨現值。
- **公式：**
	$`PI = \frac{\text{Net Present Value (NPV)}}{\text{Initial Investment (Cost)}} = \frac{PV(\text{Future Cash Flows}) - \text{Cost}}{\text{Cost}}`$
### 2. 決策規則
- 計算所有正 NPV 專案的 PI，並**由高到低進行排序**，優先選擇 PI 最高的專案，直到預算用盡。
- **限制：** PI 法則僅在**只有一個資源約束**（如僅有資金限制）且專案**不可分割**時完全成立。

## 📌 五、 經典例題與解題地圖 (Practice Questions)
### 例題 1：常規專案的 NPV 與 IRR 計算
> **題目：** 某專案初始投資 10,000 元，未來兩年分別產生現金流入 6,000 元與 7,000 元。若資金成本為 10%。
> 1. 計算專案的 NPV。
> 2. 計算專案的 IRR，並判定是否接受。
- **解答流程：**
	1. **NPV 計算：**
	   $`NPV = -10,000 + \frac{6,000}{1.10} + \frac{7,000}{1.10^2} = -10,000 + 5,454.55 + 5,785.12 = 1,239.67`$
	2. **IRR 計算：**
	   設 $-10,000 + \frac{6,000}{1+IRR} + \frac{7,000}{(1+IRR)^2} = 0$。
	   令 $x = \frac{1}{1+IRR}$，則 $7,000x^2 + 6,000x - 10,000 = 0$。
	   解二次方程得 $x \approx 0.8358 \implies 1+IRR = 1.1964 \implies IRR \approx 19.64\%$。
	   因 $IRR = 19.64\% > 10\%$，應接受該專案。
- **答案：** NPV = **1,239.67** 元；IRR = **19.64%**，接受專案。
---
### 例題 2：融資型專案的 IRR 陷阱
> **題目：** 某專案今天為你帶來 1,000 元收入，但明年你必須支付 1,200 元。資金成本為 10%。請問該專案的 IRR 與 NPV 各為多少？你該接受嗎？
- **解答流程：**
	- 這是融資型現金流：$C_0 = +1,000, C_1 = -1,200$。
	- **IRR 計算：**
	  $`1,000 - \frac{1,200}{1+IRR} = 0 \implies IRR = 20\%`$
	- **NPV 計算：**
	  $`NPV = 1,000 - \frac{1,200}{1.10} = 1,000 - 1,090.91 = -90.91`$
	- **決策分析：** 雖然 $IRR = 20\% > 10\%$，但 NPV < 0。這相當於你以 20% 的成本借錢，但市場資金成本僅為 10%。你不應該接受這個融資。
- **答案：** IRR = **20%**，NPV = **-90.91** 元。**不應該接受**該專案。
---
### 例題 3：多重 IRR 問題
> **題目：** 某環保專案初始投資 1,000 元，第 1 年帶來 2,300 元回報，但第 2 年因拆除成本需要支出 1,320 元。請問該專案有幾個 IRR？
- **解答流程：**
	- 方程：$`-1,000 + \frac{2,300}{1+r} - \frac{1,320}{(1+r)^2} = 0`$
	- 同乘 $(1+r)^2$：$`-1,000(1+r)^2 + 2,300(1+r) - 1,320 = 0`$
	- 簡化：$`10x^2 - 23x + 13.2 = 0`$ (其中 $x = 1+r$)
	- 解得 $x_1 = 1.10 \implies r_1 = 10\%$，以及 $x_2 = 1.20 \implies r_2 = 20\%$。
- **答案：** 該專案有兩個 IRR，分別為 **10%** 與 **20%**。當資本成本為 12% 時，IRR 法則失效。
---
### 例題 4：互斥專案的規模衝突 (⚠️ 常考)
> **題目：** 公司有兩個互斥專案，專案 A 需投入 10,000 元，明年回收 12,000 元；專案 B 需投入 100,000 元，明年回收 115,000 元。資金成本為 8%。
> 1. 計算兩專案的 IRR。
> 2. 計算兩專案的 NPV，並說明公司該如何選擇。
- **解答流程：**
	- **專案 A：**
	  - $IRR_A = \frac{12,000}{10,000} - 1 = 20\%$
	  - $NPV_A = -10,000 + \frac{12,000}{1.08} = 1,111.11$ 元
	- **專案 B：**
	  - $IRR_B = \frac{115,000}{100,000} - 1 = 15\%$
	  - $NPV_B = -100,000 + \frac{115,000}{1.08} = 6,481.48$ 元
- **答案：** 雖然 $IRR_A (20\%) > IRR_B (15\%)$，但公司應選擇 **專案 B**，因為它的 $NPV_B (6,481.48) > NPV_A (1,111.11)$，能為股東創造更多絕對財富。
---
### 例題 5：資本約束下的資源配置 (PI 法則)
> **題目：** 公司總預算限制為 100 萬元。有以下三個獨立專案可供選擇：
> - 專案 1：初始投資 60 萬元，NPV = 12 萬元。
> - 專案 2：初始投資 40 萬元，NPV = 10 萬元。
> - 專案 3：初始投資 50 萬元，NPV = 11 萬元。
> 請問公司該如何挑選？
- **解答流程：**
	1. 計算 PI：
	   - $PI_1 = \frac{12}{60} = 0.20$
	   - $PI_2 = \frac{10}{40} = 0.25$
	   - $PI_3 = \frac{11}{50} = 0.22$
	2. 按 PI 由高到低排序：專案 2 (0.25) > 專案 3 (0.22) > 專案 1 (0.20)。
	3. 搭配選擇：
	   - 先選專案 2（投資 40 萬，剩餘預算 60 萬）。
	   - 再選專案 3（投資 50 萬，剩餘預算 10 萬，無法再投專案 1）。
	   - 總 NPV = $10 + 11 = 21$ 萬元，總投資 = 90 萬元。
	   - (對比：若選 1 + 2，總投資 100 萬，總 NPV = $12 + 10 = 22$ 萬。此時因為專案不可分割，PI 排序法在預算剩餘較多時需要與其他組合進行總 NPV 對比確認。本題中，1+2 組合的 NPV 為 22 萬，高於 2+3 的 21 萬。因此最佳決策是選擇專案 1 與專案 2)。
- **答案：** 在專案不可分割的極端情況下，需比較組合的總 NPV，最優決策為選擇 **專案 1 與 專案 2**，創造 22 萬元的總價值。

## 📌 六、 程式化財務思維：JavaScript 決策工具 (IRR & PI)
我們可以使用數值逼近法 (Raphson-Newton 或簡單割線法) 來計算任何不規則現金流的 IRR，並依據 PI 進行專案最佳化排序：
```javascript
/**
 * 計算專案的 NPV
 */
function getNpv(cashFlows, r) {
  return cashFlows.reduce((total, cf, t) => total + cf / Math.pow(1 + r, t), 0);
}

/**
 * 估算 IRR (使用割線法 Secant Method)
 */
function calculateIrr(cashFlows) {
  let r0 = 0.05;
  let r1 = 0.15;
  const tolerance = 1e-6;
  
  for (let i = 0; i < 100; i++) {
    const npv0 = getNpv(cashFlows, r0);
    const npv1 = getNpv(cashFlows, r1);
    
    if (Math.abs(npv1 - npv0) < 1e-10) break;
    
    const rNext = r1 - npv1 * (r1 - r0) / (npv1 - npv0);
    
    if (Math.abs(rNext - r1) < tolerance) {
      return rNext;
    }
    r0 = r1;
    r1 = rNext;
  }
  return r1;
}

// === 測試 IRR 規模衝突 ===
const projectA = [-10000, 12000];
const projectB = [-100000, 115000];

console.log(`專案 A 的 IRR: ${(calculateIrr(projectA) * 100).toFixed(2)}%`); // 20.00%
console.log(`專案 B 的 IRR: ${(calculateIrr(projectB) * 100).toFixed(2)}%`); // 15.00%
```

### 2. 本章三大考試/常考觀念
1. **互斥專案的選擇核心：** 當 IRR 與 NPV 結論衝突時，**永遠以 NPV 大的為準**。因為股東要的是口袋裡「多出來的財富絕對值」，而不是回報率百分比。
2. **多重 IRR 與無 IRR：** 如果現金流方向改變多次，就不能直接用 IRR 法則。需要看 NPV 曲線。
3. **PI 的核心原理：** PI 衡量的是「單位資源的利用率」，在資本有限的條件下，它能引導公司挑選出最佳的專案組合。
