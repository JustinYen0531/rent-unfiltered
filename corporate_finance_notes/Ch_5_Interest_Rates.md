# Ch. 5: Interest Rates 利率
## 🎯 章節核心問題
> **如何正確比較不同計息頻率的貸款或投資？是什麼決定了未來的利率走勢，以及如何評估無風險之外的實際機會成本？**
>
> **核心解答：**
> 正確評估利率的關鍵在於將所有報價轉換為**實質年利率 (EAR)**。利率受到通貨膨脹的實質侵蝕（費雪效應），並會隨時間長短呈現不同的**期限結構（殖利率曲線）**。最終，財務決策所採用的折現率，必須是與該投資專案風險等同的**資金機會成本（稅後利率）**。
---
## 📌 一、 利率報價與調整 (Interest Rate Quotes & Adjustments)
在金融市場中，利率的報價方式會因為計息頻率不同而產生誤導，必須進行標準化轉換。
### 1. 實質年利率 (Effective Annual Rate, EAR)
- **定義：** 考慮複利效應後，投資人在一年內實際賺取或支付的利率。
- **特徵：** 無論複利頻率為何，EAR 是真正反映資金增長速度的客觀指標。
### 2. 名目年利率 (Annual Percentage Rate, APR)
- **定義：** 不考慮複利效應，單純將每期利率乘以一年的計息期數所得到的利率。
- **特徵：** APR 不能直接當作折現率使用！因為它忽略了期中複利的影響。
- **公式關係：**
	$`\text{每期利率 (Interest Rate per Period)} = \frac{APR}{m}`$
	其中 $m$ 為一年內的計息期數（如每月複利 $m=12$，每季複利 $m=4$）。
### 3. APR 與 EAR 轉換公式
若已知 APR，且一年複利 $m$ 次，則 EAR 為：
$`1 + EAR = \left( 1 + \frac{APR}{m} \right)^m`$
$`EAR = \left( 1 + \frac{APR}{m} \right)^m - 1`$
若已知 EAR，想反推 APR：
$`APR = m \times \left[ (1 + EAR)^{1/m} - 1 \right]`$
- **連續複利 (Continuous Compounding) 極限：**
	當計息期數 $m \to \infty$ 時：
	$`EAR = e^{APR} - 1`$
	$`APR = \ln(1 + EAR)`$

## 📌 二、 通貨膨脹與利率：名目 vs. 實質 (Inflation & Real Interest Rates)
### 1. 名目利率 (Nominal Interest Rate)
- 銀行或債券上所標明、未扣除通貨膨脹影響的貨幣利率。
### 2. 實質利率 (Real Interest Rate)
- 扣除通貨膨脹購買力損耗後的實際購買力增長率。
### 3. 費雪效應 (The Fisher Effect)
名目利率 ($r_n$)、實質利率 ($r_r$) 與預期通貨膨脹率 ($i$) 之間的精確關係為：
$`1 + r_r = \frac{1 + r_n}{1 + i}`$
整理後可得實質利率的精確計算公式：
$`r_r = \frac{r_n - i}{1 + i}`$
- **近似公式 (常規低通膨環境下適用)：**
	$`r_r \approx r_n - i`$
---
## 📌 三、 利率的期限結構與殖利率曲線 (Term Structure & Yield Curve)
### 1. 期限結構 (Term Structure of Interest Rates)
- 不同到期期限（如 1 年、5 年、10 年、30 年）的無風險無息債券（如美國國庫券）利率之間的關係。
### 2. 殖利率曲線 (Yield Curve)
- 將不同期限的利率繪製成圖所得到的曲線。一般有三種形態：

## 📌 四、 稅收與風險對利率的影響 (Taxes & Risk)
### 1. 稅後利率 (After-Tax Interest Rate)
利息收入通常需要課稅。因此，投資人真正獲得的是稅後報酬率，借款人實際承擔的也是稅後借款成本。
- **公式：**
	$`r_{\text{after-tax}} = r \times (1 - \tau)`$
	其中 $\tau$ 為適用稅率。
### 2. 信用風險與風險溢酬
- 債務人如果存在違約可能性，投資人會要求高於無風險利率的風險補貼。
- 信用差的公司發行債券時，必須支付更高的殖利率（高收益債/垃圾債）。
---
## 📌 五、 資金的機會成本 (Opportunity Cost of Capital)
- **定義：** 將資金投資於當前專案所放棄的其他「同等風險水平」投資計畫的預期回報率。
- **財務決策鐵律：** 評估 any 投資專案的現值時，折現率必須採用該專案的**資金機會成本**，而不是隨機決定的利率，也不是取得資金的表面利息。

## 📌 六、 經典例題與解題地圖 (Practice Questions)
### 例題 1：APR 與 EAR 轉換
> **題目：** 某信用卡宣稱其年利率 (APR) 為 18%，按月複利。請問該信用卡實際的年利率 (EAR) 是多少？
- **解答流程：**
	1. 確定計息期數 $m = 12$。
	2. 每月利率為 $\frac{APR}{12} = \frac{18\%}{12} = 1.5\% = 0.015$。
	3. 套用公式：
	   $`EAR = (1 + 0.015)^{12} - 1`$
	   $`EAR = 1.1956 - 1 = 19.56\%`$
- **答案：** 該信用卡實際年利率 (EAR) 約為 **19.56%**。
---
### 例題 2：不同複利頻率下的投資比較
> **題目：** 銀行 A 提供 6.0% 的年利率，按半年複利；銀行 B 提供 5.9% 的年利率，按每月複利。請問你該選擇哪家銀行存錢？
- **解答流程：**
	- **銀行 A：** $m = 2$
	  $`EAR_A = \left( 1 + \frac{0.06}{2} \right)^2 - 1 = (1.03)^2 - 1 = 6.09\%`$
	- **銀行 B：** $m = 12$
	  $`EAR_B = \left( 1 + \frac{0.059}{12} \right)^{12} - 1 \approx (1.004917)^{12} - 1 = 6.06\%`$
- **答案：** 應該選擇 **銀行 A**，因為其實質年利率 (6.09%) 高於銀行 B (6.06%)。
---
### 例題 3：費雪效應計算實質利率
> **題目：** 某銀行的一年期定期存款名目利率為 5%，而當年的預期通貨膨脹率為 3%。請問該存款的實質利率是多少？
- **解答流程：**
	- 精確公式：
	  $`r_r = \frac{r_n - i}{1 + i} = \frac{0.05 - 0.03}{1 + 0.03} = \frac{0.02}{1.03} \approx 1.942\%`$
	- 近似公式：
	  $`r_r \approx 5\% - 3\% = 2\%`$
- **答案：** 精確的實質利率為 **1.94%** (近似為 2.0%)。
---
### 例題 4：稅後利率與投資選擇
> **題目：** 你面臨兩個投資機會：一是免稅的市政債券，殖利率為 4.5%；二是需要繳稅的公司債券，殖利率為 6.0%。如果你的所得稅率為 30%，你該選哪一個？
- **解答流程：**
	- 市政債券稅後收益率 = 4.5%（免稅）
	- 公司債券稅後收益率 = $6.0\% \times (1 - 0.30) = 4.2\%$
- **答案：** 應該選擇 **市政債券**，因為其稅後回報率 (4.5%) 高於公司債券的稅後回報 (4.2%)。
---
### 例題 5：非年計息期下的年金計算 (⚠️ 高頻考題)
> **題目：** 你計劃買一輛車，貸款 30,000 元，期限為 4 年，貸款利率 (APR) 為 6% 按月複利。請問你每個月底應支付多少分期付款？
- **解答流程：**
	- 貸款是按月支付，因此必須使用**月折現率**和**月期數**來計算普通年金。
	1. 月折現率 $r = \frac{APR}{12} = \frac{6\%}{12} = 0.5\% = 0.005$。
	2. 總期數 $N = 4 \times 12 = 48$ 個月。
	3. 套用普通年金公式：
	   $`30,000 = \frac{C}{0.005} \left[ 1 - \frac{1}{(1.005)^{48}} \right]`$
	   $`30,000 = C \times 42.5803`$
	   $`C = \frac{30,000}{42.5803} \approx 704.55`$
- **答案：** 每個月底應還款 **704.55** 元。

## 📌 七、 程式化財務思維：JavaScript 利率轉換工具
在實際專案中，我們可以用程式碼進行計息頻率的精確調整：
```javascript
/**
 * 將 APR 轉換為 EAR
 * @param {number} apr - 名目年利率 (小數表示, 如 0.06)
 * @param {number} m - 一年內的複利次數
 * @returns {number} 實質年利率 EAR
 */
function convertAprToEar(apr, m) {
  return Math.pow(1 + apr / m, m) - 1;
}

/**
 * 將 EAR 轉換為 APR
 * @param {number} ear - 實質年利率 (小數表示, 如 0.0618)
 * @param {number} m - 目標複利次數
 * @returns {number} 名目年利率 APR
 */
function convertEarToApr(ear, m) {
  return m * (Math.pow(1 + ear, 1 / m) - 1);
}

// === 測試範例 ===
const myApr = 0.06; // 6% APR
const monthlyEar = convertAprToEar(myApr, 12);
console.log(`6% APR 每月複利相當於 EAR: ${(monthlyEar * 100).toFixed(4)}%`);
// 輸出: 6% APR 每月複利相當於 EAR: 6.1678%

const targetApr = convertEarToApr(monthlyEar, 4);
console.log(`該 EAR 對應的每季複利 APR 為: ${(targetApr * 100).toFixed(4)}%`);
// 輸出: 該 EAR 對應的每季複利 APR 為: 6.0302%
```

### 2. 本章三大考試/常考觀念
1. **APR 絕不能直接拿來折現：** 凡是題目給 APR 且為非年計息（如半年、按月），第一步永遠是將 APR 除以 $m$ 得到該期利率，或者先轉成 EAR 後再做年金計算。
2. **殖利率曲線倒掛的解讀：** 正常情況下長期利率大於短期利率，若短期利率大於長期利率即為倒掛，是預測經濟衰退的最關鍵指標。
3. **機會成本是同等風險下的回報：** 折現率不是公司自己隨意定的，而是該項目的同等風險替代投資方案的稅後預期報酬率。
